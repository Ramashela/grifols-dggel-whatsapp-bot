# Grifols DG Gel WhatsApp AI Bot

A simple AI-powered WhatsApp chatbot that helps lab technicians with:

- Product information
- Step-by-step instructions
- Troubleshooting
- DG Gel system guidance

All answers are strictly based on the official Grifols DG Gel catalogue.

---

## 🚀 Features

- WhatsApp Cloud API integration
- PDF-based knowledge (RAG)
- Local vector database (FAISS)
- No paid database required
- Beginner-friendly setup
